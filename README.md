Proyek ini merupakan sistem manajemen Posyandu (Pos Pelayanan Terpadu) berbasis web dan mobile yang dirancang untuk mendukung digitalisasi layanan kesehatan ibu dan anak. Sistem ini memadukan CMS edukasi publik, Kartu Menuju Sehat (KMS) digital, dashboard monitoring, serta prediksi risiko stunting berbasis Machine Learning.

Aplikasi ditujukan untuk digunakan oleh kader posyandu, kepala posyandu, staf, serta masyarakat umum, dengan fokus pada kemudahan penggunaan, keamanan data, dan pengambilan keputusan berbasis data.
